<p>A Minecraft Project that adds crafting recipes for normally uncraftable items in Survival mode, along with custom advancements for a more rewarding experience.</p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>‚ú® <strong>Features</strong></p>
<p>* <strong>New Crafting Recipes</strong>: Craft items like the Enchanted Golden Apple, Totem of Undying, Trident, Horse Armors, Chainmail Armor, and more.</p>
<p>* <strong>Custom Advancements</strong>: Unique rewards for crafting special items.</p>
<p>* <strong>Wide Compatibility</strong>: Supports Minecraft Java Edition from version 1.13.x up to the latest snapshots.</p>

<p><strong>Installation</strong> üóÇÔ∏è</p>
<p>Linux:</p>
<p>&nbsp;</p>
<p>World that already exists:</p>
<p>1. Go to this location "~/.minecraft/saves/" (Make sure you are showing the Hidden Folders and Files by pressing Left Ctrl + H).</p>
<p>2. Choose the world folder you want to put the Data Pack in it.</p>
<p>3. Open datapacks folder, then put the data pack.zip in the datapacks folder.</p>
<p>4. Reload the world by /reload command or by Quit the world and rejoin it.</p>
<p>&nbsp;</p>
<p>New World:</p>
<p>1. Open Minecraft, then click on "Singleplayer".</p>
<p>2. Click on "Create New World", then click "More World Options...".</p>
<p>3. Click on "Data Packs" button, then drag the data pack file into Minecraft.</p>
<p>4. Click "Done" and "Create New World".</p>
<hr>

<p>1. Press Win + R, then type %appdata%, then press Enter.</p>
<p>2. Open "Roaming" folder, then ".minecraft" folder, then open "saves" (Make sure you are showing the Hidden Folders and Files by pressing Left Ctrl + Left Shift + . (period)).</p>
<p>4. Choose the world folder you want to put the Data Pack in it.</p>
<p>5. Open datapacks folder, then put the data pack.zip in the datapacks folder.</p>
<p>6. Reload the world by /reload command or by Quit the world and rejoin it.</p>
<p>&nbsp;</p>
<p>New World:</p>

<p>2. Click on "Create New World", then click "More World Options...".</p>
<p>3. Click on "Data Packs" button, then drag the data pack file into Minecraft.</p>
